GMLRVA - Generic Multiple Layout Recycler View Adapter
===============
[ ![Download](https://api.bintray.com/packages/simdea/GMLRVA/gmlrva.lib/images/download.svg) ](https://bintray.com/simdea/GMLRVA/gmlrva.lib/_latestVersion)
